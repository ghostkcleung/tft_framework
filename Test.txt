WGS84经纬度转换为大地坐标系中的经纬度算法如下：

1. 将WGS84经纬度转换为WGS84椭球体坐标系的笛卡尔坐标系。WGS84椭球体的长半轴为a=6378137米，短半轴为b=6356752.3142米，扁率为f=1/298.257223563。

x = (N + h) * cos(lat) * cos(lon)
y = (N + h) * cos(lat) * sin(lon)
z = (N * (1 - e^2) + h) * sin(lat)

其中，N为卯酉圈曲率半径，e为椭球体第一偏心率，h为海拔高度。

N = a / sqrt(1 - e^2 * sin(lat)^2)
e^2 = 2f - f^2

2. 将WGS84椭球体坐标系的笛卡尔坐标系转换为大地坐标系的笛卡尔坐标系。大地坐标系使用的椭球体参数为a=6378137米，b=6356752.3141米，扁率为f=1/298.257223563。

X = (1 + k) * x
Y = (1 + k) * y
Z = (1 + k - e^2) * z

其中，k为大地坐标系的比例因子，可以通过以下公式计算：

k = (a * e^2) / (sqrt(X^2 + Y^2) * sqrt(1 - e^2 * sin(lat)^2))

3. 将大地坐标系的笛卡尔坐标系转换为大地坐标系的经纬度。

lon = atan(Y / X)
lat = atan(Z / sqrt(X^2 + Y^2))

其中，atan为反正切函数，返回值为弧度制。需要将结果转换为度数制。

需要注意的是，以上算法中的经纬度坐标系为弧度制，需要将结果转换为度数制。此外，由于计算过程中涉及到三角函数的计算，可能会存在精度误差。
